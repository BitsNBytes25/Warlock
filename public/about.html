About Warlock Express App

This is a simple Express.js application that demonstrates serving HTML pages.

Features:

  • Express.js server setup
  • Static file serving from public directory
  • Multiple HTML pages with routing
  • Basic CSS styling
  • Development server with nodemon support

Getting Started:

  1. Run npm install to install dependencies
  2. Run npm start to start the production server
  3. Run npm run dev to start the development server with auto-reload
